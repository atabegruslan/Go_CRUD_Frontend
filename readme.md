# Install

- Install Go
- Install Beego framework: `go get github.com/astaxie/beego`
- Install Bee tool: `go get github.com/beego/bee`

# Create project

`bee new beegoing_front`

# Run

1. `cd beegoing_front`
2. `bee run`
3. Visit http://localhost:8083/place
